  • 제작기간3주
  • 사용 프로그램HTML / CSS / JavaScript / jQuery

#반응형 #SCSS_적용

  • SCSS를 활용하여 처음 작업한 웹사이트입니다. SCSS를 배우면서 기존 CSS보다 작성하기 편리하고 여러 구성에 대한 효율적인 관리 등 좋은 장점들을 알게 됐습니다. 처음에는 설치해야하는 편집기 플러그인도 있고 SCSS를 위한 여러 파일의 준비가 까다롭다고 느꼈지만 막상 해보니 관리하기도 편하고 특히 반응형 제작 시 기존 CSS 미디어쿼리 작성보다 직관적으로 만들 수 있었던 것이 정말 장점으로 느껴졌습니다. 무엇보다 주제선정에 있어 내가 좋아하는 아티스트의 페이지를 가상으로 작업해봐서 더욱 재밌었던 SCSS 프로젝트입니다.

  • 제작기간4일
  • 사용 프로그램HTML / SCSS / jQuery
